This is a tool I designed to help me when facing clients that want to redefine the selection of pictures they want me to display on an event's room screens. 

Context:
When a client comes to me (lighting-laser-video guy) to check if the picture render well on the room's screens, it often happens they decide not to display some pictures or want to add some new ones to the "to display" list. 
Some clients want them looping in order, others want them shuffling, ...

Being on windows, when the client asks to see the pictures, I usually used windows' build-in media viewer and used the slideshow tool. But when the client asks to remove a picturer, I then have to memorise the picture's name, exit the slideshow, search for the file, delete the file, then reload the slideshow, and jump to the picture we last saw. 
This takes a lot of time, and if I'm speaking to a production assistant then they will likely already be taking care of another problem in the meantime, and I'll be there sitting ducks waiting for her to come back. 
Also if I want to be certain not to have to redownload the send pictures, I'd have to prepare a separate 'initial' folder in case I have to get a deleted picture back.

So this tool is:
1) A file manager that clones the media
2) A file viewer able to take in many formats, and also to mix pictures and videos

Existing features:
1) File Manager
  - Add files
  - Remove files from list
  - Play/Pause slidehsp
2) Viewer
  - Play/Pause
  - Next
  - Shuffle/Ordered
  - Fullscreen
  - Deals with multiple formats
  - Deals with images AND videos
  - Set interval before media change if it is an image
